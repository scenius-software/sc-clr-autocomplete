Observable Autocomplete Model

Explanation

Demonstrates the ability to show a loading spinner while waiting for an observable to return an array of strings to use as our autocomplete model. The demo below has an observable that delays for 5 seconds before returning its list of strings. During this period, a loading spinner will be displayed. For this, we'll be using the ObservableAutocompleteModel<T> object which expects an observable that returns T. This model will show a loading spinner while the observable hasn't completed yet.
