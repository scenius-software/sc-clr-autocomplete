HTTP Autocomplete Model

Explanation

Demonstrates the ability to use a URL to parse an array of items from a REST API. In this example, the call is used: https://restcountries.eu/rest/v2/all. For this we will make use of the HttpAutocompleteModel, which we provide with a HttpClient and an url to make a request on our behalf. We also provide a custom displaySelector in order to tell the component how to render the returned objects. This model will show a loading spinner while the HTTP call has not yet resolved.
