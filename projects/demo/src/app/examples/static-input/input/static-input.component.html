Static Autocomplete Model

Explanation

Demonstrates the basic functionality of the autocomplete input, using a static string array initializer. As such, the returned value will be the free input value of the user.

The snippets below demonstrate the simplest way of using the autocomplete component. The component works by providing it a model which is a fancy term for the data source that will be queried based on the user input. There are various models available, all of them demonstrated on this page. This example demonstrates the simplest model, the StaticAutocompleteModel<string>
